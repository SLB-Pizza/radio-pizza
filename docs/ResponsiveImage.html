# <ResponsiveImage />

Creates a JSX <picture> element with responsive sources that ImageHelper calls upon.

Parameters:
Name Type Description
mainUrl String

the Imgix optimized Prismic image URL for the large image size; bulma "fullhd"
responsiveData Object

an object containing the Imgix optimized Prismic image URLs for the responsive sizes

Name Type Description

data object to make the image for bulma "widescreen" breakpoint

data object to make the image for bulma "tablet" breakpoint
responsiveData.mobile String

data object to make the image for bulma "mobile" breakpoint
